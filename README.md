## Epidemic Online Course Management System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/1bc3ad2b42cf43db8d4a585b360f0a17/ghb20250916) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular Student, Teacher, Administrator.

**Regular User Functions:**
1. Login
2. Register
3. Homepage Information
4. View Course Schedule Information
5. Forum Communication, Posting, Comments
6. View School Announcements
7. Personal Center
8. My Posts
9. My Favorites
10. My Course Information
11. Submit Assignments
12. Submit Leave Requests
13. My Leave Requests
14. Class Attendance
15. My Attendance Information

**Teacher Functions:**
1. Login
2. Register
3. Personal Center
4. View Student Information
5. View Classes
6. View Course Categories
7. View Course Schedule
8. Course Information Management
9. Review Assignment Information
10. Review Leave Request Information
11. View Class Attendance


**Administrator Functions:**
1. Login
2. Personal Center
3. Student Management
4. Teacher Management
5. Class Management
6. Course Category Management
7. Course Schedule Management
8. Course Information Management
9. Assignment Information Management
10. Leave Request Information Management
11. Class Attendance Management
12. Forum Communication
13. Carousel Management
14. School Announcements
